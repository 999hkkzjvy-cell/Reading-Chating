# 以读攻独 · 会员积分、票券与徽章系统说明

欢迎加入「以读攻独」读书会！本文档帮助你了解会员成长体系的运作方式——如何获得贡献值、票券有什么用途、徽章如何解锁。

---

## 一、会员身份

| 身份 | 说明 |
|---|---|
| **游客** | 未登录，可浏览公开内容标题与摘要 |
| **基础会员** | 已注册用户，可发布阅读动态、累积贡献值 |
| **成长会员** | 根据贡献值处于 Lv.1 ~ Lv.16 |
| **管理员** | 站点管理者，负责内容与权限运营 |
| **开创者** | 独立权限，全站资源永久开放 |

> 开创者是一种独立权限，不是 Lv.17。如果你是开创者，你仍然有自己的会员等级和成长徽章，开创者权限是你额外拥有的全站通行资格。

---

## 二、会员等级与段位

会员等级根据**贡献值**自动计算，无需手动申请。每当你获得新的贡献值，系统会自动判断你是否升级。

| 段位 | 对应等级 | 每周资源浏览券 |
|---|---|---|
| 基础会员 | 注册后，Lv.1 之前 | 0 张 |
| 青铜会员 | Lv.1 ~ Lv.6 | 1 张 |
| 白银会员 | Lv.7 ~ Lv.11 | 2 张 |
| 黄金会员 | Lv.12 ~ Lv.16 | 3 张 |
| 开创者 | 独立权限 | 不限 |

### 等级升级

升级时，系统会自动为你发放：

- **对应等级成长徽章** 1 枚
- **共读兑换券** 1 张

## 三、贡献值

贡献值是会员成长的核心指标。你留下的阅读痕迹越多、质量越高，贡献值增长越快。

### 如何获得贡献值

| 行为 | 贡献值 | 说明 |
|---|---|---|
| 发布公开阅读动态 | +5 | 每日最多 3 条计入 |
| 动态绑定豆瓣链接 | +2 | 填写有效豆瓣链接 |
| 感想达到 20 字 | +5 | 与发布基础分叠加 |
| 感想达到 100 字 | 额外 +5 | 即 20 字 +5 后，满 100 字再加 5 |
| 完成一本书（在读 → 已读） | +30 | 从标记在读变为已读 |
| 收到点赞 | 每赞 +1 | 每日上限 20 贡献值 |

### 贡献值回收

以下情况将回收对应贡献值：

- 删除阅读动态
- 将公开动态改为仅自己可见
- 管理员隐藏或删除内容
- 取消点赞（回收点赞贡献值）

> 贡献值回收可能导致等级下降。如果发生降级，该等级升级时发放但**尚未使用**的共读兑换券将被回收。已使用的兑换券不受影响。

### 动态发布示例

> 你发布了一条公开阅读动态，填写了书名和豆瓣链接，写了 150 字的感想。
>
> **本次获得：** +5（发布公开动态）+ 2（绑定豆瓣）+ 5（满 20 字）+ 5（满 100 字）= **17 贡献值**

## 四、票券系统

「以读攻独」有三种票券，用途各不相同。

### 1. 资源浏览券

**用途：** 临时解锁一个受保护资源，解锁后 **72 小时**内可重复查看。

**获取方式：**

| 方式 | 数量 |
|---|---|
| 首次注册 | 1 张 |
| 青铜会员每周发放 | 1 张 |
| 白银会员每周发放 | 2 张 |
| 黄金会员每周发放 | 3 张 |
| 本周活跃榜前 10 名 | 下周数量翻倍 |

**有效期：** 发放后 7 天内有效，过期作废。

**使用方式：** 浏览受保护资源时，系统会提示你是否消耗 1 张资源浏览券临时解锁。

### 2. 共读纪念券

**用途：** 证明你参与了当期共读，并**永久解锁**本期共读资源。

**获取方式：**

1. 点击「想要共读」并进入共读群
2. 管理员在群内提供本期共读密码
3. 在站内输入密码，填写你的群内 ID/昵称
4. 验证通过后，自动发放共读纪念券并永久解锁资源

> 每期共读密码独立，同一用户同一期只能领取一次。
>
> 领取时会记录你的群内昵称，用于核对社群参与情况。

### 3. 共读兑换券

**用途：** 兑换**任一期历史共读资源**的永久浏览权限。

**获取方式：**

- 每次会员等级升级发放 1 张

**使用方式：** 在资源权限页面，选择你想兑换的历史共读资源，消耗 1 张共读兑换券。

**注意事项：**

- 兑换后不可撤销
- 兑换历史资源**不会**获得该期的共读纪念徽章
- 如果等级降级，该等级升级时发放但未使用的兑换券将被回收

## 五、资源权限

部分共读资源（如会议链接、延伸读物、干货正文）属于参与共读的权益内容。

### 你能看到什么

无论你的权限如何，你始终可以看到：

- 资源所属的分页名称
- 资源标题
- 摘要或简介
- 资源数量
- 解锁方式提示

### 权限判断顺序

当你点击受保护资源时，系统按以下顺序判断：

1. **管理员？** → 直接开放
2. **开创者？** → 直接开放
3. **拥有该书永久权限？**（共读纪念券/共读兑换券解锁过）→ 直接开放
4. **拥有该资源有效临时权限？**（72 小时内用资源浏览券解锁过）→ 直接开放
5. **有可用资源浏览券？** → 弹出确认框，询问是否消耗 1 张解锁
6. **无权限** → 显示标题、摘要和获取方式提示

> **提示语（无权限时）：**
> 这部分是共读参与权益。你可以使用资源浏览券临时查看，也可以通过参与共读、提升贡献值获得共读兑换券，解锁完整资源。

## 六、徽章系统

徽章是你阅读旅程的见证。目前系统包含 **17 枚徽章**：16 枚等级成长徽章 + 1 枚开创者徽章。

### 徽章类型

| 类型 | 说明 |
|---|---|
| **成长徽章**（Lv.1 ~ Lv.16） | 达到对应等级时自动获得 |
| **开创者徽章** | 拥有开创者权限时展示 |
| **纪念徽章**（后续开放） | 参与特定共读活动获得 |

### 在哪里查看

进入**会员中心**即可查看：

- 当前佩戴的等级徽章
- 已获得的所有徽章列表
- 未解锁徽章的解锁条件

### 后续计划

未来版本将推出**徽章谜面答题**玩法——点击未解锁徽章会显示一段文学谜面，答对即可提前解锁或获得额外贡献值。

## 七、阅读动态

阅读动态是你留下的阅读足迹，也是贡献值的主要来源。

### 可发布的内容类型

| 类型 | 说明 |
|---|---|
| 想读 | 标记一本想读的书 |
| 在读 | 标记正在阅读的书 |
| 已读 | 完成一本书的阅读 |
| 摘抄/短评 | 书中的精彩片段或简短点评 |
| 书评/共读笔记 | 深度书评或共读心得（长内容） |

### 发布规则

- 每日最多 **3 条**阅读动态计入贡献值（超出可发布但不计分）
- 可选择**公开**或**仅自己可见**
- 仅自己可见的内容不计贡献值
- 修改可见范围会重新计算贡献值

### 动态广场

在动态广场，你可以：

- 浏览所有会员的公开阅读动态
- 为喜欢的动态点赞
- 筛选最新、我的动态、同本书的动态

## 八、常见问题

**Q：贡献值可以购买吗？**

A：不可以。贡献值完全由你的阅读行为和社群参与产生，不涉及付费。

**Q：等级降级后已兑换的资源会被收回吗？**

A：不会。已使用的票券兑换结果保留。只有未使用的票券会被回收。

**Q：资源浏览券过期了怎么办？**

A：每周固定发放新的资源浏览券。活跃参与共读、提升等级可以获得更多。

**Q：如何成为开创者？**

A：开创者由管理员授予，或通过管理员发放的特殊邀请码激活。

**Q：共读密码在哪里获取？**

A：加入当期共读群后，管理员会在群内公布。请在站内输入密码并填写群内昵称完成验证。
